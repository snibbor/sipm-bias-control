Items needing your eyes on the schematic

  • U2 (LT1761ES5-SD#TRMPBF): 20V Vin tolerance. If U2 sits on a rail >20V, this substitute won't survive. The TP_1.22V1 testpoint suggests U2 produces a 1.22V reference from a low-voltage input (5V or 3.3V), which is fine — but confirm.
  • U3 (MCP6C02T-050E/CHY): Gain changed from 20 to 50 V/V. Firmware needs to multiply the ADC reading by 0.4 to keep current calibration consistent. If firmware is auto-scaling already (e.g. via a gain-config register), no work. Otherwise, one constant in the firmware needs to change.
  • C12 (Samsung CL10C220JB8NNNC): Value changed 25pF → 22pF because Samsung doesn't make 25pF/0603/C0G at DigiKey. ±5pF on LT8362 FB compensation is electrically negligible. If you want exactly 25pF, ask for a non-Samsung swap.
  • R6 (52.3k): Assumes LT8362 build per README. For LT8361 build, swap to 39k.
  • IC2 (ATSAMD21E18A-AFT): Automotive grade per your schematic (T&R variant of -AF for qty-1 ordering).
  • C5/C6/C8/C11/C13 (CL31B225KCHSNNE, 2.2µF/100V/X7R/1206): X7R derates 50-70% at 70V DC bias. Consider 1210/1812 if loop margin is tight.
